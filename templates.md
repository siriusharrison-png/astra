# Templates

## Purpose

This file stores structured templates for repeated work.

## Template Types

- spec draft
- decision record
- discussion summary
- review checklist
- transition note
- task review to convergence note
- convergence output note

## Template Rules

- Templates should be short enough to reuse quickly
- Templates should reflect the path used for the task
- Templates should make the next action obvious
- Templates should not duplicate stable rules
- Templates should be written in English first

## Template Rule

Do not add templates until the structure and standards are stable.

## Usage Rule

Use this template at the end of a task when there is enough context to decide what should be promoted, revised, archived, or ignored.
